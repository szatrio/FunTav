## Installation

### Clone
```bash
$ git clone https://github.com/szatrio/funtav

Back-End
$ cd funtav_backend
$ npm install

Front-End
$ cd funtav_frontend
$ npm install
```
---

### Create Environment Variable
```bash
$ cp .env.example .env
$ nano .env
```
---
### Start Development Server
```bash
$ npm start
```
